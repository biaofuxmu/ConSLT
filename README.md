# ConSLT (ICASSP2023)

This repo contains code for the paper [A Token-level Contrastive Framework for Sign Language Translation](https://arxiv.org/abs/2204.04916). 

## Data Preparation

Download the feature files. 

`sh data/download.sh`

## Training and Inference

  `sh train.sh`
